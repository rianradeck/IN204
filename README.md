# IN204

## Formatter
Using vscode cpp extension formatter with the `.clang-format` file (Microsoft based formatter)

## Running
```ps1
.\run.ps1 td_path
```
`td_path` is optional. If omitted, the script will compile and execute code.cpp from the root dir.
